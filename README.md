# Offline vectorkaart

Plaats hier later een echte compacte kaart, bijvoorbeeld:

- `nederland.pmtiles`
- of een ander offline vectorbestand

V18 gebruikt nu nog `data/offline_nederland_simple.geojson` als lichte offline kaart. De editor/routes gebruiken GPS-coördinaten, dus jouw eigen vaarroutes en bruggen blijven op dezelfde plek wanneer de achtergrond later wordt vervangen door PMTiles.
